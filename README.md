 # TP React - JSX, Composition et Tests
 ---
📋 Description
---
Ce TP pratique vous permettra de maîtriser les concepts fondamentaux de React :
---
JSX et sa transformation
---
Les Higher-Order Components (HOC)
---
Le pattern Render Props
---
Les tests avec Jest et Testing Library
---
🚀 Installation

---

# Installer les dépendances
---
npm install

# Lancer l'application
---
npm start

---

src/

│   └── Counter.test.js      # Test intégration

├── App.css                    # Styles
